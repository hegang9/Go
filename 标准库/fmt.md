# fmt 标准库详细指南

`fmt` 包实现了类似 C 语言 `printf` 和 `scanf` 的格式化 I/O。它主要分为向外输出（Print）和获取输入（Scan）两大部分。

## 常用 API 说明

### 1. `fmt.Println` / `fmt.Print`
* **功能**：将参数列表按照默认格式输出到标准输出（控制台）。`Println` 会自动在元素之间添加空格，并在末尾添加换行符；`Print` 不会加换行符和自动空格。
* **参数说明**：`a ...any`（任意类型的不定参数）。
* **注意事项**：不要用于高性能日志打印，它涉及反射和锁，性能一般。
* **适用场景**：日常调试程序、打印简单结果信息。

### 2. `fmt.Printf`
* **功能**：根据指定的占位符格式化输出字符串到标准输出。
* **参数说明**：
  * `format string`：格式化控制字符串（如 `%s`, `%d`, `%v` 等）。
  * `a ...any`：对应占位符的变量。
* **高频占位符**：
  * `%v`：默认格式的值；`%+v`：打印结构体时会包含字段名；`%#v`：Go 语法表示的值。
  * `%T`：打印变量的类型。
  * `%s`：字符串；`%d`：十进制整数。
* **注意事项**：传入的参数个数和类型必须与占位符匹配，否则会输出 `%!(MISSING)` 错误。
* **适用场景**：需要按特定格式排版输出内容时。

### 3. `fmt.Sprintf`
* **功能**：与 `Printf` 类似，但它不将结果打印到控制台，而是将格式化后的最终内容作为**字符串返回**。
* **参数说明**：同 `Printf`。
* **注意事项**：是 Go 中拼接字符串常用的方式之一，但在极高性能要求的循环中拼接长字符串，推荐使用 `strings.Builder`。
* **适用场景**：生成带有动态参数的 SQL 语句、构造日志信息、生成特定格式的返回值。

### 4. `fmt.Errorf`
* **功能**：根据格式化说明符格式化字符串，并返回一个 `error` 类型的对象。
* **参数说明**：同 `Printf`。
* **注意事项**：Go 1.13 起支持 `%w` 占位符，用于错误嵌套（Error Wrapping），方便后续通过 `errors.Is` 和 `errors.As` 进行判断。
* **适用场景**：返回带上下文信息的错误（如：`fmt.Errorf("查询数据库失败, userID: %d, err: %w", id, err)`）。

### 5. `fmt.Scan` / `fmt.Scanf` / `fmt.Scanln`
* **功能**：从标准输入读取文本并将用空格分隔的值保存到传递的参数中。
* **参数说明**：传入对应类型的**指针**。
* **注意事项**：读取整行或含空格的字符串时容易出错，更推荐使用 `bufio.Scanner`。
* **适用场景**：简单的交互式命令行工具获取用户输入。
