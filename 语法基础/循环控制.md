# Go 语言循环控制

Go 语言在设计上非常简洁，它**没有** `while` 和 `do-while` 循环。Go 语言中唯一的基础循环结构是 `for` 语句，但它的功能非常强大，可以组合出各种循环模式。

## 1. for 循环的三种基本形式

### 1.1 经典的 for 循环 (类似 C 语言)
包含初始化语句、条件表达式和后置语句。

```go
for init; condition; post {
    // 循环体代码
}
```
**示例：**
```go
for i := 0; i < 5; i++ {
    fmt.Println(i)
}
```
*注意：`init` 和 `post` 语句是可选的，并且 `for` 后面的表达式不需要用小括号 `()` 括起来，但循环体必须用大括号 `{}` 括起来。*

### 1.2 省略 init 和 post 语句 (充当 while 循环)
如果只保留条件表达式，`for` 循环就变成了其他主流语言中的 `while` 循环。

```go
n := 1
for n < 5 {
    n *= 2
    fmt.Println(n)
}
```

### 1.3 省略所有条件 (死循环)
如果省略所有的语句，就构成了一个无限循环。通常配合 `break` 语句使用。

```go
for {
    // 无限循环
    fmt.Println("This is an infinite loop!")
    break // 退出循环
}
```

## 2. for...range 循环 (遍历)

`for...range` 是 Go 语言中极其常用的语法，用于遍历数组 (Array)、切片 (Slice)、映射 (Map)、字符串 (String) 或通道 (Channel)。

### 2.1 遍历数组/切片
可以同时获取索引 (index) 和对应的元素值 (value)。
```go
nums := []int{10, 20, 30}
for index, value := range nums {
    fmt.Printf("索引: %d, 值: %d\n", index, value)
}
```
如果你只需要值并不关心索引，可以使用**空白标识符** `_` 忽略索引：
```go
for _, value := range nums {
    fmt.Println("值:", value)
}
```

### 2.2 遍历 Map
可以同时获取键 (key) 和值 (value)。
```go
kvs := map[string]string{"a": "apple", "b": "banana"}
for k, v := range kvs {
    fmt.Printf("键: %s, 值: %s\n", k, v)
}
```

### 2.3 遍历字符串
遍历字符串时，`range` 会按 Unicode 字符（rune）进行遍历，而不是按固定的字节，这对于包含中文的字符串非常友好。
```go
for index, runeValue := range "Go语言" {
    fmt.Printf("索引: %d, 字符: %c\n", index, runeValue)
}
```

## 3. 循环控制语句

Go 语言提供了 `break`、`continue` 和 `goto` 来控制循环的执行流程。

### 3.1 break 语句
用于强制退出**当前整个**循环代码块。
```go
for i := 1; i <= 5; i++ {
    if i == 3 {
        break // 当 i=3 时退出整个循环
    }
    fmt.Println(i) // 输出 1, 2
}
```

### 3.2 continue 语句
用于跳过**当前这一次**循环迭代，直接进入下一次迭代。
```go
for i := 1; i <= 5; i++ {
    if i == 3 {
        continue // 跳过 i=3 的情况
    }
    fmt.Println(i) // 输出 1, 2, 4, 5
}
```

### 3.3 配合标签(Label)的高级控制
`break` 和 `continue` 后面可以跟着一个标签(Label)，用于跳出或继续**指定层级**的外层循环，常用于多重循环控制。
```go
OuterLoop:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if i == 1 && j == 1 {
            break OuterLoop // 直接跳出名为 OuterLoop 的整个外层循环
        }
        fmt.Printf("i: %d, j: %d\n", i, j)
    }
}
```

### 3.4 goto 语句
无条件地转移到程序中指定的标签行。建议尽量少在业务代码中大量使用，容易造成逻辑混乱。
```go
func main() {
    for i := 0; i < 3; i++ {
        if i == 1 {
            goto End
        }
        fmt.Println(i)
    }
End:
    fmt.Println("直接跳到了这里！")
}
```
