# Go语言字面量 (Literals)

字面量是指由字母、数字等构成的字符串或者数值，它只能作为右值出现。在Go语言中，字面量包括整型字面量、浮点数字面量、复数字面量、字符字面量和字符串字面量。

## 1. 整型字面量 (Integer Literals)

整数可以使用不同的进制表示。在Go 1.13及以上版本中，增加了一些新的前缀以及数字分隔符（`_`）。

| 进制 | 前缀 | 示例 | 说明 |
| :--- | :--- | :--- | :--- |
| 十进制 (Decimal) | 无 | `42`, `1_000` | 最常用的整数表示，第一位不能是 `0`（除非是 `0` 本身）。 |
| 二进制 (Binary) | `0b` 或 `0B` | `0b1010`, `0B11_00` | 只能包含 `0` 和 `1`。 |
| 八进制 (Octal) | `0` 或 `0o` / `0O` | `0755`, `0o755` | `0` 后跟 `0-7` 的数字。推荐使用 `0o` 以增加可读性。 |
| 十六进制 (Hex) | `0x` 或 `0X` | `0xFF`, `0x1A_2B` | 包含 `0-9` 和 `a-f` (或 `A-F`)。 |

**数字分隔符**：可以使用下划线 `_` 来分隔数字，以提高可读性。分隔符不能作为前缀或后缀，只能在前缀符号之后以及数字之间使用。

## 2. 浮点数字面量 (Floating-point Literals)

浮点数包含整数部分、小数点、小数部分和指数部分。支持十进制和十六进制格式。

| 类型 | 格式 | 示例 | 说明 |
| :--- | :--- | :--- | :--- |
| 标准十进制 | 数学小数 | `3.14`, `.5`, `1.` | 可以省略整数部分或小数部分。 |
| 科学记数法 | 用 `e` 或 `E` 表示 | `1e6`, `6.022e23`, `1E-3` | 表示基数为 10 的指数。 |
| 十六进制浮点数 | 用 `0x` + `p` 或 `P` 表示 | `0x1p-2`, `0x2.p10` | 表示基数为 2 的指数（Go 1.13 新增）。 |

*注：浮点数同样支持使用下划线 `_` 作为数字分隔符，如 `3.141_592_653`。*

## 3. 复数字面量 (Complex Literals)

复数包含实部和虚部，虚部是以数字后跟加上小写字母 `i` 来表示。

* **示例**：`0i`, `2.5i`, `1e3i`, `3 + 4i` (实际上是整型/浮点型字面量和复数字面量的表达式组合)

## 4. 字符字面量 (Rune Literals)

在Go语言中，字符常量通常被称为 Rune（符文），它是 `int32` 的别名。字符字面量由一对单引号包裹。

* **普通字符**：`'a'`, `'中'`, `'1'`
* **Unicode 码位**：可以使用 `\u`（4位十六进制）或 `\U`（8位十六进制）表示。
  * `\u6C49` (代表 '汉')
  * `\U00006C49`
* **八进制或十六进制转义**：
  * `\xff` (2位十六进制)
  * `\377` (3位八进制)

## 5. 转义字符 (Escape Sequences)

在字符（Rune）或双引号字符串中，可以使用反斜杠 `\` 进行转义。常用的转义字符如下表：

| 转义字符 | 说明 |
| :--- | :--- |
| `\a` | 响铃 (Bell) |
| `\b` | 退格 (Backspace) |
| `\f` | 换页 (Form feed) |
| `\n` | 换行 (New line) |
| `\r` | 回车 (Carriage return) |
| `\t` | 制表符 (Tab) |
| `\v` | 垂直制表符 (Vertical tab) |
| `\\` | 反斜杠本身 |
| `\'` | 单引号 (主要在字符字面量中使用) |
| `\"` | 双引号 (主要在字符串字面量中使用) |

## 6. 字符串字面量 (String Literals)

Go 语言支持两种类型的字符串字面量格式：解释型和原生型。

| 类型 | 符号 | 特点 | 示例 |
| :--- | :--- | :--- | :--- |
| **解释型字符串** | 双引号 `""` | 支持转义字符（如 `\n`），通常在一行内完成。 | `"Hello, Go!\n"` |
| **原生型字符串** | 反引号 <code>` `</code> | 不支持转义序列，保留所有格式（包含换行），内容保持原样输出。 | <code>`原生\n字符串`</code> |

**原生多行字符串示例**：
```go
var str = `第一行
第二行
第三行`
```
