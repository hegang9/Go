## 标识符命名规则
Go 语言中，标识符（变量名、函数名、类型名等）的命名规则如下：
- 只能以字母、数字、下划线组成
- 只能以字母或下划线开头
- 严格区分大小写
- 包内标识符不能重复
- 不能与关键字冲突

## 运算符
#### 运算符优先级
```go
Precedence    Operator
    5             *  /  %  <<  >>  &  &^
    4             +  -  |  ^
    3             ==  !=  <  <=  >  >=
    2             &&
    1             ||
    0             <-  =  +=  -=  *=  /=  %=  &=  |=  ^=  <<=  >>=  &^=
```

#### 取反运算符
Go 语言中取反运算符为 `^`，它有两种用法：
1. 位取反：对一个整数的每一位进行取反操作，即将0变为1，1变为0。例如，`^5` 的结果是 `-6`，因为5的二进制表示是 `0000 0101`，取反后是 `1111 1010`，即-6。
2. 位清除：`a &^ b` 表示将 a 中与 b 中对应位为1的位清除掉。例如，`5 &^ 3` 的结果是 `4`，因为5的二进制表示是 `0000 0101`，3的二进制表示是 `0000 0011`，清除后是 `0000 0100`，即4。

#### 自增自减运算符
与C++不同，Go 语言中的自增（`++`）和自减（`--`）运算符只能作为语句使用，不能作为表达式的一部分，且**只能在变量右边**。例如，`i++` 是合法的，但 `j = i++` 是不合法的，会导致编译错误。**即`i++`没有返回值**。
