## 变量
### 变量声明
在Go中，变量类型声明后置，且需要使用var关键字，格式为`var 变量名 变量类型`
```go
var a int   // 单个变量声明
var a, b int // 批量变量声明
var (
  name    string
  age     int
  address string
)   // 批量变量声明，每个变量都可以有不同的类型
```
### 变量赋值
在Go中，变量赋值使用`=`运算符，格式为`变量名 = 表达式`
```go
var a int
a = 10
var b int
a,b = 10,20
```

以上几种赋值方式每次都要指定类型，可以使用官方语法糖`短变量初始化`来省略var和返回类型：
```go
name := "张三"
age := 18
address := "中国"
```
变量具体类型交给编译器推断，但在后续赋值时，类型必须保持一致

注意：:= 短变量初始化只能在初始化时使用，不能在后续赋值时使用。但有一个例外，那就是在赋值旧变量的同时声明一个新的变量
```go
a := 1
a, b := 2, 2
```

### 局部变量
GO语言有一个限制，所有局部变量都必须被使用，不允许声明后未被使用（编译错误）。

### 匿名
用下划线表示不需要某个变量。未使用的变量是无法通过编译的，当你不需要某一个变量时，就可以使用下划线_代替，避免未使用变量导致编译错误。

### 交换
```go
num1, num2, num3 := 25, 36, 49
num1, num2, num3  = num3, num2, num1
```
注：**先计算值再赋值，不是从左到右顺序执行**。

### 比较
**比较的前提：变量类型完全相同。Go语言中不存在隐式类型转换。**
```go
func main() {
  var a uint64
  var b int64
  fmt.Println(a == b)   // 编译错误
  fmt.Println(int64(a) == b)    // 使用强制类型转换
}   
```

### 可比较类型
Go语言中可比较类型有：
- bool
- 数字
- 字符串
- 指针
- 通道（仅支持相等比较）
- 元素是可比较类型的数组（切片不可比较）（仅支持判断是否相等）（仅支持相同长度的数组间的比较，因为数组长度也是类型的一部分，而不同类型不可比较）
- 字段类型是可比较类型的结构体（仅支持相等比较）
