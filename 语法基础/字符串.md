# Go 语言字符串 (String)

在 Go 语言中，字符串是一个**不可变**的字节序列。Go 语言的字符串默认使用 **UTF-8** 编码，这意味着它可以非常精简和高效地处理包含中文在内的任意 Unicode 字符。

## 1. 字符串的声明与字面量

声明字符串一般有两种方式：使用双引号 `""` 或 反引号 ` `` `。

### 1.1 双引号 `""` (解释型字符串)
支持转义字符（如 `\n`, `\t`, `\"` 等）。这是最常用的形式。
```go
s1 := "Hello, Go!"
s2 := "包含\n换行符"
```

### 1.2 反引号 ` `` ` (原生字符串)
不支持任何转义字符。所有的可见和不可见字符（包括换行符）都会原样输出，常用于编写多行文本（如 HTML 模板、JSON 字符串或复杂的正则表达式）。
```go
s3 := `第一行
第二行
不会转义 \n`
```

## 2. 字符串的不可变性

Go 语言中的字符串是**不可变的 (Immutable)**。一旦字符串被创建，就无法修改其中的某一个字符。

```go
s := "hello"
// s[0] = 'H' // 编译错误：cannot assign to s[0]
```
如果要修改字符串内容，必须先将其转换为切片（`[]byte` 或 `[]rune`），修改完成后再转换回字符串：
```go
s := "hello"
bytes := []byte(s) // 转换为字节切片 (适用于纯 ASCII)
bytes[0] = 'H'
s = string(bytes)  // "Hello"
```

## 3. String、Byte 和 Rune

因为 Go 的字符串默认包含的是 UTF-8 编码的字节，在处理中文字符时必须区分字节（Byte）和字符（Rune）。

- **`byte`**：是 `uint8` 的别名。代表一个单字节（常用于 ASCII 字符）。
- **`rune`**：是 `int32` 的别名。代表一个 Unicode 码点（Code Point，即一个真实的字符，如一个汉字）。

### 3.1 长度计算 (len vs RuneCount)
内置函数 `len()` 获取的是字符串的**字节数**，而不是字符数。对于中文字符，在 UTF-8 编码下通常占用 3 个字节。
```go
s := "Go语言"
fmt.Println(len(s)) // 输出: 8 (2 个英文字母字节 + 2 个汉字 * 3 字节 = 8)

// 要获取真实的字符数，需要使用 unicode/utf8 包
import "unicode/utf8"
fmt.Println(utf8.RuneCountInString(s)) // 输出: 4
```

### 3.2 字符串遍历
遍历字符串时，不同的方式会导致不同的结果：
```go
s := "Go语言"

// 1. 按照字节 (byte) 遍历
for i := 0; i < len(s); i++ {
    fmt.Printf("%x ", s[i]) 
    // 汉字会被截断输出
}

// 2. 按照字符 (rune) 遍历，强烈推荐这种方式，可以正确处理中文！
for index, char := range s {
    fmt.Printf("位置: %d, 字符: %c\n", index, char)
}
```

## 4. 字符串切片 (Slicing)

可以通过切片语法从字符串中截取子串。由于切片针对的是**底层字节**，因此切分包含中文的字符串时如果不小心，可能会切断汉字的 UTF-8 编码，导致出现乱码（）。

```go
s := "Hello, World"
sub1 := s[0:5] // "Hello"
sub2 := s[7:]  // "World"
```

如果需要截取带中文的字符串，建议先转成 `[]rune`：
```go
s := "Go语言编程"
runes := []rune(s)
sub := string(runes[0:4]) // "Go语言"
```

## 5. 字符串拼接与常用操作

### 5.1 字符串拼接
1. **使用 `+` 运算符**：简单直观，但由于字符串不可变，每次拼接都会分配新内存，适合少量拼接。
2. **使用 `fmt.Sprintf`**：常用于需要格式化不同数据类型的拼接。
   ```go
   s := fmt.Sprintf("%s 是最好的 %d 门语言之一", "Go", 1)
   ```
3. **使用 `strings.Builder` (推荐)**：针对大量拼接场景，性能最高。
   ```go
   import "strings"
   var builder strings.Builder
   builder.WriteString("Hello,")
   builder.WriteString(" World")
   s := builder.String()
   ```

### 5.2 `strings` 标准库
Go 提供了一个强大的 `strings` 包用来处理常见的字符串操作：
```go
import "strings"

// 包含子串
strings.Contains("hello", "ll") // true

// 统计子串出现次数
strings.Count("cheese", "e") // 3

// 字符串前缀/后缀
strings.HasPrefix("http://...", "http") // true
strings.HasSuffix("test.go", ".go")     // true

// 字符串分割与组合
parts := strings.Split("a,b,c", ",")         // []string{"a", "b", "c"}
joined := strings.Join([]string{"a", "b"}, "-") // "a-b"

// 大小写转换
strings.ToUpper("hello") // "HELLO"
strings.ToLower("HELLO") // "hello"
```
