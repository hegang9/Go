## 函数

函数是Go语言中的基本构建块，用于执行特定任务。函数可以接受参数并返回结果。

### 函数声明

```go
// func 关键字表示函数，函数参数类型后置，返回值类型也后置
func functionName(parameters type) returnType {
    // 函数体
}

// 函数可以有多个返回值
func functionName(parameters type) (returnType1, returnType2) {
    // 函数体
}

// 命名返回值
func getName()(name string,age int,isActive bool){
    name = "Alice"
    age = 30
    isActive = true

    return // 直接返回命名的返回值
}

// 可变参数函数
func sum(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

// 多种类型参数
func printInfo(prefix string, values ...interface{}) {
    fmt.Print(prefix + ": ")
    for _, v := range values {
        fmt.Printf("%v ", v)
    }
    fmt.Println()
}

// 第二种声明函数的方法
var add = func(a, b int) int {
    return a + b
}
```

### 函数调用

```go
functionName() // 无参函数调用
result := functionName(arguments)   // 带参函数调用，arguments 是传递给函数的参数、

result1,result2 := functionName(arguments) // 多返回值函数调用

result3, _ := functionName(arguments) // 如果不需要某个返回值，可以使用空白标识符 `_` 来忽略它

sum(1, 2, 3, 4) // 可变参数函数调用
printInfo("Info", "Alice", 30, true) // 多种类型参数函数调用

// 匿名函数调用，理解为 C++ 中的匿名函数对象
result := func(a, b int) int {
    return a + b
}(5, 3) // 直接调用匿名函数，传递参数

result2 := func(a, b int) int {
    return a * b
} // 定义匿名函数并赋值给变量 result2
fmt.Println(result2(5, 3)) // 调用匿名函数
```

### 函数作为参数传递

在 Go 语言中，函数是一等公民（First-class citizen），这意味着函数可以像普通变量一样被赋值、作为参数传递给其他函数，或者作为其他函数的返回值。这种接收函数作为参数或返回函数的函数被称为**高阶函数**。

```go
// 定义一个接收操作函数作为参数的高阶函数
func compute(a int, b int, op func(int, int) int) int {
    return op(a, b)
}

func add(x, y int) int {
    return x + y
}

func subtract(x, y int) int {
    return x - y
}

func main() {
    // 将函数 add 和 subtract 作为参数传递
    res1 := compute(10, 5, add)      // res1 = 15
    res2 := compute(10, 5, subtract) // res2 = 5
}
```

### 闭包 (Closures)

匿名函数通常会与**闭包**结合使用。闭包是一个函数值，它引用了其函数体之外的变量。该函数可以访问并赋予其引用的变量的值，换句话说，该函数被这些变量“绑定”在一起。

```go
// intSeq 函数返回另一个在函数体内部定义的匿名函数。
// 返回的匿名函数闭带了变量 i，形成了一个独立的内部状态。
func intSeq() func() int {
    i := 0
    return func() int {
        i++
        return i
    }
}

func main() {
    nextInt := intSeq()
    fmt.Println(nextInt()) // 1
    fmt.Println(nextInt()) // 2
    fmt.Println(nextInt()) // 3

    // 重新调用会创建一个全新的独立闭包状态
    newInts := intSeq()
    fmt.Println(newInts()) // 1
}
```

### defer 语句 (延迟执行)

`defer` 语句用于安排一个函数在当前函数**返回之前（执行完毕前最后时刻）**自动执行。它非常适合用来处理资源清理工作状态（如关闭文件、关闭数据库连接、释放锁等）。多个 defer 语句按照**后进先出 (LIFO)** 的栈顺序执行。

```go
func readFile() {
    fmt.Println("打开文件")
    // 无论函数因为什么原因返回，这行代码都会在退出前最后执行
    defer fmt.Println("关闭文件") 

    fmt.Println("读取文件数据")
    // 实际输出顺序：打开文件 -> 读取文件数据 -> 关闭文件
}
```

### 注意事项：
1. **左括号位置**：Go 语言中函数体的左大括号 `{` 必须与函数声明在同一行，不能换行，否则会导致编译错误，这是由于 Go 编译器自动插入分号的机制决定的。
2. **不支持函数重载 (Overloading)**：Go 不允许在同一作用域内定义多个同名但参数列表不同的函数，这使得代码更加简单清晰。
3. **不支持默认参数**：Go 的函数参数不能有默认值。如果需要类似功能，通常通过可变参数 `...` 或传递配置结构体（Options Pattern）来实现。
4. Go语言中的函数参数传递是**值传递**，这意味着函数接收的是参数的副本。如果需要在函数内部修改外部变量的值，可以使用指针或返回修改后的值。但是对于切片和 map 这类引用类型，虽然参数传递是值传递，但它们内部包含的指向底层数据结构的指针会被复制，因此在函数内部修改切片或 map 的内容会影响到外部变量。

## 匿名函数
匿名函数是没有名字的函数，可以直接定义并调用。它们常用于需要一个临时函数来完成某个任务的场景，或者作为高阶函数的参数。

```go
// 定义并立即调用一个匿名函数
result := func(a, b int) int {
    return a + b
}(5, 3) // 直接传递参数并调用匿名函数
fmt.Println(result) // 输出: 8
```
