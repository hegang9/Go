# Go语言结构体 (Struct)

Go 语言中没有传统面向对象语言（如 Java、C++、Python）中的“类 (Class)”概念。Go 通过**结构体 (Struct)** 来实现面向对象编程中的属性封装和数据聚合。结构体是由一系列具有相同类型或不同类型的字段（数据）构成的数据集合。

## 1. 结构体的定义
使用 `type` 和 `struct` 关键字来定义一个结构体：
```go
type Person struct {
    Name   string
    Age    int
    Gender string
}
```

## 2. 结构体的实例化（初始化）

### 2.1 声明变量 (默认全为零值)
当声明一个结构体变量但未赋予初值时，其内部所有字段都将采用与其类型对应的零值（例如 `int` 为 `0`，`string` 为 `""`，布尔为 `false`）。
```go
var p1 Person
p1.Name = "Alice"
p1.Age = 25
```

### 2.2 键值对初始化（最常用、最推荐）
可以指定具体的字段名来进行初始化。未被点名的字段会自动取零值。
```go
p2 := Person{
    Name: "Bob",
    Age:  30,
    // Gender 缺失，默认为 ""
}
```

### 2.3 值的列表初始化
必须**严格按照结构体字段定义的顺序**来进行初始化，且**不能省略**任何字段。如果后期结构体增加了新属性，采用这种初始化的代码都会报错。
```go
p3 := Person{"Charlie", 20, "Male"}
```

### 2.4 获取结构体指针
可以使用 `new` 关键字，或者在字面量初始化前加 `&` 来获取一个指向该结构体的指针。在 Go 语言底层，对于结构体指针，可以直接使用 `.` 来接取属性（编译器会自动进行 `(*p).Name` 这样的解引用操作）。
```go
p4 := new(Person) // 返回的是 *Person 指针，内部字段全为零值
p4.Name = "David" // 底层等价于 (*p4).Name = "David"

p5 := &Person{
    Name: "Eve",
    Age:  28,
} // 返回指针，带初始化数据，这是工程中最常见的写法
```

## 3. 匿名结构体
如果一个结构体对象仅作为临时一次性的数据载体（比如临时构建一个请求参数对象），不需要在别处复用该类型定义，可以使用匿名结构体：
```go
func main() {
    // 定义的同时直接初始化
    msg := struct {
        Code int
        Msg  string
    }{
        Code: 200,
        Msg:  "Success",
    }
    fmt.Println(msg.Code, msg.Msg)
}
```

## 4. 结构体的嵌套与组合（面向对象中的继承）
Go 语言虽然没有 `extends` 来实现继承，但是它巧妙地通过**嵌套和组合**来实现代码与属性的复用。

当一个结构体中嵌套了另一个匿名的结构体时，内部结构体的字段将被直接提升 (Promoted) 到外部结构体上。

```go
type Address struct {
    City     string
    Province string
}

type User struct {
    Name    string
    Age     int
    Address // 匿名嵌套（嵌入字段）：直接写类型名，字段名同时也是 Address
}

func main() {
    u := User{
        Name: "Tom",
        Age:  18,
        Address: Address{
            City:     "Shenzhen",
            Province: "Guangdong",
        },
    }
    
    // 提升字段 (Promoted fields)：可以直接通过外部实例访问内部匿名嵌套对象的字段
    fmt.Println(u.City)     // 实际上是调用的 u.Address.City
    fmt.Println(u.Province) // 实际上是调用的 u.Address.Province
}
```

## 5. 结构体标签 (Struct Tags)
结构体标签是非常强大的特性，它常常依附在字段定义之后。主要用于提供该字段在进行特殊映射（如 JSON 序列化/反序列化、XML 映射、数据库 ORM 映射）时的规则元数据。它是通过反射 (`reflect`) 机制生效的。

```go
import "encoding/json"

type Student struct {
    // JSON 转换时字段变为小写的 "id"，数据库映射时的列名为 "student_id"
    ID   int    `json:"id" db:"student_id"` 
    Name string `json:"name"`
    // "-" 表示在 JSON 序列化/反序列化时彻底忽略该字段，通常用于密码、敏感配置等
    Password string `json:"-"` 
    // "omitempty" 指定如果该字段的值为空值（如0、空字符串、nil指针等），在序列化时不仅不显示空值，而是直接连同该键一起去掉。
    Age  int    `json:"age,omitempty"`
}
```

## 6. 结构体与方法 (Method)
在 Go 中，你可以为任意你自定义的类型（主要就是结构体）绑定**函数**，这就构成了**方法 (Method)**。方法只不过是带有**接收者 (Receiver)** 的函数。

接收者有两种：
- **值接收者 (`T`)**：调用方法时，传递的是结构体的拷贝。在方法内对结构体字段修改，**不会影响**原对象。
- **指针接收者 (`*T`)**：调用方法时，传递的是结构体指针。在方法内对字段的修改，**会直接影响**原对象。（推荐用法，特别是当对象内存占用巨大，或必须修改其状态时）。

```go
type Circle struct {
    Radius float64
}

// 这是一个绑定了 Circle 对象的方法 -- 值接收者
func (c Circle) Area() float64 {
    return 3.14 * c.Radius * c.Radius
}

// 这是一个绑定了 Circle 对象的方法 -- 指针接收者
func (c *Circle) SetRadius(r float64) {
    c.Radius = r // 由于是指针接收者，外界对象的 Radius 确实会被改变
}
```
