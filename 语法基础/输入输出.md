# Go语言输入输出核心概念

Go 语言在操作系统层面抽象了三个主要的文件描述符变量，由 `os` 标准库提供：
- `os.Stdin`：标准输入（通常关联键盘）
- `os.Stdout`：标准输出（通常关联终端屏幕）
- `os.Stderr`：标准错误输出

围绕它们，Go 提供了多种不同级别的 I/O 方法，主要由 `内置函数`、`os`、`fmt`、`bufio` 几个部分提供。

---

## 1. 输出 (Output)

### 1.1 内置函数 print / println
Go中有两个内置函数 `print` 和 `println`，用于最简单的输出（其中 `println` 附带换行）。
* **注意**：它们主要用于调试阶段，生产环境中不推荐使用，因为它们会将结果输出到标准错误（Standard Error），且不支持复杂的格式化。
```go
package main

func main() {
    print("hello world!")
    println("hello world!")
}
```

### 1.2 os 标准库 (底层 I/O)
可以直接使用 `os.Stdout` 提供的方法将数据（`[]byte` 或 `string`）写入标准输出。
```go
package main

import "os"

func main() {
    // 写入字节切片
    os.Stdout.Write([]byte("hello byte!\n"))
    // 写入字符串
    os.Stdout.WriteString("hello world!\n")
}
```

### 1.3 fmt 包 (最常用的格式化 I/O)
`fmt` 包是对底层 `os` 标准库接口的高级封装，功能强大，可读性高，但由于内部使用反射，在极高频且性能极其敏感的场景下要慎用。

常见的三大家族：
* **Print 系列**（默认输出至标准输出）：`Print`、`Println`（带换行）、`Printf`（格式化）。
* **Fprint 系列**（输出至指定的 `io.Writer`）：`Fprint`、`Fprintln`、`Fprintf`。可以输出到文件、网络或标准输出/错误。
* **Sprint 系列**（不输出，而是将结果返回为字符串）：`Sprint`、`Sprintln`、`Sprintf`。

```go
package main

import (
    "fmt"
    "os"
)

func main() {
    fmt.Print("直接打印 ")
    fmt.Println("打印并换行")
    fmt.Printf("格式化打印: %s\n", "Golang")

    // 定向输出到标准错误
    fmt.Fprintln(os.Stderr, "这是一条错误信息")
    
    // 生成字符串变量
    msg := fmt.Sprintf("生成的字符串: %d", 100)
    fmt.Println(msg)
}
```

### 1.4 bufio (带缓冲的高效 I/O)
`bufio` 提供了可缓冲的输出方法，数据会先写入内存（默认 4KB），积累到阈值后再真正进行系统级的底层写入。
* **适用场景**：高频、大量的文件或网络等 I/O 写入，避免频繁系统调用导致的性能下降。

```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    writer := bufio.NewWriter(os.Stdout)
    writer.WriteString("hello ")
    fmt.Fprintf(writer, "formatted %s!\n", "world")
    
    // 必须调用 Flush 才能将缓冲区的数据真正推入底层的 Writer（这里是标准输出）
    writer.Flush() 
}
```

## 2. 输入 (Input)

### 2.1 os 标准库
使用 `os.Stdin.Read` 可以从标准输入中读取最底层的数据流（字节流）。通常需要自行处理边界和换行。
```go
package main

import "os"

func main() {
    buf := make([]byte, 1024)
    n, err := os.Stdin.Read(buf)
    if err != nil {
        panic(err)
    }
    // 打印读取到的字节数及内容
    println("Read bytes:", n)
    print("Content: ", string(buf[:n]))
}
```

### 2.2 fmt 包
提供基于空格分隔以及指定格式读取的功能：`Scan`、`Scanf`、`Scanln`。
* **`Scan`**：从标准输入扫描，以空格为分隔符连续读取。
* **`Scanln`**：类似 `Scan`，但在遇到换行符时停止读取。
* **`Scanf`**：根据格式化字符串来解析输入。

```go
package main

import "fmt"

func main() {
    var name string
    var age int
    fmt.Print("请输入您的名字和年龄 (空格分隔): ")
    // 注意：需要传入变量的指针
    fmt.Scanf("%s %d", &name, &age)
    fmt.Printf("Name: %s, Age: %d\n", name, age)
}
```

### 2.3 bufio 的 Reader 与 Scanner (推荐用于读取整行)
用 `fmt` 读取含有空格的完整一行非常费劲。通常更推荐 `bufio.Scanner` 或 `bufio.Reader`。
```go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    // 方式一：使用 Scanner (最简单，以换行为界)
    fmt.Print("请输入一行文字：")
    scanner := bufio.NewScanner(os.Stdin)
    if scanner.Scan() {
        line := scanner.Text()
        fmt.Println("您输入的是:", line)
    }

    // 方式二：使用 Reader (更底层，可指定自定义分隔符)
    fmt.Print("请再输入一行文字：")
    reader := bufio.NewReader(os.Stdin)
    lineStr, err := reader.ReadString('\n')
    if err == nil {
        fmt.Println("您又输入了:", lineStr)
    }
}
```

## 3. 格式化 (Formatting)

Go 中的格式化输出/输入功能（特别是 `fmt.Printf` 和 `fmt.Sprintf`）依赖于“格式化动词”（Formatting Verbs）。

| 动词 | 说明 | 示例 |
| :--- | :--- | :--- |
| **通用** | | |
| `%v` | 默认格式输出，打印该值的普遍表示（结构体打印 `%+v` 会包含字段名） | `fmt.Printf("%v", user)` |
| `%#v` | Go 语法表示的值，此形式可以直接当作对应的 Go 代码 | `fmt.Printf("%#v", user)` |
| `%T` | Go 语法表示的类型 | `fmt.Printf("%T", 100)` -> `int` |
| `%%` | 字面上的百分号 | `fmt.Printf("100%%")` -> `100%` |
| **布尔值** | | |
| `%t` | 布尔值，`true` 或 `false` | `fmt.Printf("%t", true)` |
| **整数** | | |
| `%b` | 二进制表示 | `fmt.Printf("%b", 5)` -> `101` |
| `%c` | 相应的 Unicode 码点所表示的字符 | `fmt.Printf("%c", 65)` -> `A` |
| `%d` | 十进制表示 | `fmt.Printf("%d", 10)` -> `10` |
| `%o` | 八进制表示 | `fmt.Printf("%o", 10)` -> `12` |
| `%O` | 八进制表示，带有 0o 前缀 | `fmt.Printf("%O", 10)` -> `0o12` |
| `%x` | 十六进制表示，使用小写字母 a-f | `fmt.Printf("%x", 15)` -> `f` |
| `%X` | 十六进制表示，使用大写字母 A-F | `fmt.Printf("%X", 15)` -> `F` |
| `%q` | 单引号（字符）或双引号（字符串）括起来的 Go 语法安全的转义表示 | `fmt.Printf("%q", "Hi")` -> `"Hi"` |
| `%U` | Unicode 格式：U+1234，等同于 "U+%04X" | `fmt.Printf("%U", '喵')` |
| **浮点数 & 复数** | | |
| `%e` / `%E` | 科学计数法（小写 e / 大写 E） | `fmt.Printf("%e", 10.2)` -> `1.020000e+01` |
| `%f` / `%F` | 有浮点数的小数部分但无指数部分 | `fmt.Printf("%.2f", 10.236)` -> `10.24` |
| `%g` / `%G` | 根据实际情况采用 `%e` / `%f` 格式，以获得更简洁的输出 | `fmt.Printf("%g", 10.200)` -> `10.2` |
| **字符串 & 字节切片** | | |
| `%s` | 直接输出字符串或 `[]byte` | `fmt.Printf("%s", "golang")` |
| **指针** | | |
| `%p` | 指针地址的十六进制表示，带前缀 0x | `fmt.Printf("%p", &a)` -> `0xc0000180a8` |
