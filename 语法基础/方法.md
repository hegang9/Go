# Go 语言基础：方法 (Methods)

在 Go 语言中，没有传统面向对象语言（如 Java、C++）中“类 (Class)”的概念。Go 语言通过给类型增加“方法 (Method)”来实现类似面向对象的行为。

简单的说，**方法就是带有接收者（Receiver）的函数**。

## 1. 方法的声明与基本语法

方法的声明和普通函数类似，只是在 `func` 关键字和方法名之间多了一个**接收者参数**。

```go
func (接收者变量 接收者类型) 方法名(参数列表) (返回参数) {
    // 方法体
}
```

- **接收者变量**：类似于面向对象语言中的 `this` 或 `self`。在 Go 中，官方建议使用接收者类型的第一个小写字母，例如 `User` 类型的接收者通常命名为 `u`。
- **接收者类型**：可以是结构体类型（最常见）或者是自定义的非指针类型。

### 示例代码

```go
package main

import "fmt"

// 定义一个结构体
type Person struct {
    Name string
    Age  int
}

// 为 Person 定义一个方法
func (p Person) SayHello() {
    fmt.Printf("Hello, my name is %s and I am %d years old.\n", p.Name, p.Age)
}

func main() {
    p1 := Person{Name: "Alice", Age: 25}
    // 调用方法：通过变量.方法名() 的方式调用
    p1.SayHello()
}
```

## 2. 值接收者与指针接收者

在定义方法时，接收者既可以是一个**值（Value）**类型，也可以是一个**指针（Pointer）**类型。这是 Go 方法中极其重要的一个概念。

### 2.1 值接收者
当且仅当不需要修改接收者内部的数据，或者接收者本身比较小（如基本类型），一般使用值接收者。
使用值接收者时，在方法内部对接收者的修改**不会**影响到调用者，因为它是值的拷贝。

```go
func (p Person) ChangeNameValue(newName string) {
    p.Name = newName // 这里的修改对外部的 p 无效
}
```

### 2.2 指针接收者
当需要修改接收者内部的数据，或者接收者是一个占用内存较大的结构体，为了避免拷贝带来的性能开销，应使用指针接收者。

```go
// 注意接收者类型变成了 *Person
func (p *Person) ChangeNamePointer(newName string) {
    p.Name = newName // 这里的修改会直接改变外部结构体的值
}
```

**什么时候必须使用指针接收者？**
1. 方法需要修改接收者的成员值。
2. 接收者是一个大型的结构体对象，拷贝的内存和时间开销较大。
3. 保持一致性：如果某个类型的某个方法使用了指针接收者，那么其所有方法通常都应该使用指针接收者。

## 3. 为基本类型添加方法

Go 语言中，不仅可以给结构体添加方法，还能给**任何自定义类型**（非内置类型）添加方法。我们不能直接给内置的基础类型（如 `int`, `string` 等）定义方法，但可以通过 `type` 关键字为其起别名，然后再定义方法。

```go
package main

import "fmt"

// 为基本类型 int 定义一个新的类型 MyInt
type MyInt int

// 为自定义类型 MyInt 定义方法
func (m MyInt) IsPositive() bool {
    return m > 0
}

func main() {
    var num MyInt = 10
    fmt.Println(num.IsPositive()) // 输出: true
}
```

*注意：你只能为同一个包中定义的类型定义方法。你不能为其他包中定义的类型（如标准库里的 `int` 或 `[]int` ）直接添加方法。*

## 4. 函数与方法的区别

| 对比项 | 函数 (Function) | 方法 (Method) |
| :--- | :--- | :--- |
| **归属** | 不归属于任何类型，是独立的程序块 | 归属于某个具体的类型（通过接收者绑定） |
| **调用方式** | `函数名(参数)` | `变量名.方法名(参数)` |
| **底层差异**| 需要严格区分参数到底是值还是指针 | Go 编译器在调用方法时会自动对值或指针进行相互转换，比如可以用值类型的变量直接调用指针接收者的方法。 |
