### 批量声明常量
```go
const (
    Pi = 3.14
    E  = 2.718
    Phi = 1.618
)

// 在同一个常量分组中，在已经赋值的常量后面的常量可以不用赋值
const (
    A = 1
    B // 等价于 B = 1
    C // 等价于 C = 1
)
const (
    A = iota // 0
    B        // 1
    C        // 2
)
const (
   Num = iota*2 // 0
   Num1 // 2
   Num2 // 4
   Num3 // 6
   Num4 // 8
)
const (
   Num = iota << 2*3 + 1 // 1
   Num1 // 13
   Num2 // 25
   Num3 = iota // 3
   Num4 // 4
)
```

### iota 常量生成器
`iota` 是 Go 语言中的一个常量生成器，常用于批量声明无类型整数常量。它在每个 `const` 块中从 0 开始自动递增，每当遇到一个新的 `const` 块时，`iota` 会重新从 0 开始计数。

### 枚举
Go 语言中没有单独的枚举类型，一般通过`自定义类型+const+iota`来实现枚举
```go
type Season uint8

const (
    Spring Season = iota
    Summer
    Autumn
    Winter
)
```

枚举实际上是数字，Go 不支持直接将它们转换为字符串，需要手动实现转换函数。
```go
func (s Season) String() string {
  switch s {
  case Spring:
    return "spring"
  case Summer:
    return "summer"
  case Autumn:
    return "autumn"
  case Winter:
    return "winter"
  }
  return ""
}
```

但是这样实现枚举有以下缺点
1. 类型不安全，因为Season是自定义类型，可以通过强制类型转换将其他数字也转为该类型
2. 繁琐，字符串表现形式需要自己实现
3. 表达能力弱，因为const仅支持基本数据类型，所以这些枚举值也只能用字符串和数字来进行表示
